# Prompt Dojo — Module 01

## Working Theme
Prompt foundations: role, objective, output, constraints.

## Module Goal
Teach a user how to write a structurally sound prompt rather than a vague request.

## First Exercise
### Task
Write a prompt that defines an AI assistant whose job is to help a project manager plan meetings.

### Optimal Reference Prompt
Describe the role of an AI assistant specifically responsible for planning meetings for a project manager, and include three concrete actions it must perform.

## Evaluation Criterion
The prompt is successful if:
- the role is clear
- the domain is specific
- the task is concrete
- the expected output includes three actions

## Feedback Template
- **What works:** identify the specific role or constraint that is present
- **What is missing:** identify vagueness, missing action count, or missing domain
- **How to improve:** rewrite with clearer role, goal, and output requirements

## Why This Is the Right First Module
It teaches structure before sophistication.
