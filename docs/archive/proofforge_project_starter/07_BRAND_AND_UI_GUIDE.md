# Brand and UI Guide

## Brand Name
ProofForge

## Positioning Tone
- precise
- capable
- restrained
- elegant
- technical without looking sterile

## Visual Direction
- no blue-led identity
- dark neutrals, warm grays, graphite, off-white, muted green or bronze accents if needed
- sharp hierarchy, low clutter
- premium but not glossy

## UI Rules
- mobile-first
- compact but breathable spacing
- no gratuitous gradients
- no dashboard bloat
- typography must do most of the work
- cards should emphasize promise, business value, and proof

## Avoid
- neon AI aesthetics
- over-animated landing pages
- crowded tag clouds
- vague motivational copy
