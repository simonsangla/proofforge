# Glossary

## Artifact
A publishable proof unit inside ProofForge.

## Lab
The portfolio index and exploration surface.

## Play
A public interactive demo route.

## Replica
A limited reproducible version of an artifact.

## Private Engine
The hidden logic, service, or system that carries the actual moat.

## Proof
Evidence that a capability exists and is operational.

## Signal
The degree to which an artifact increases trust and commercial credibility.
