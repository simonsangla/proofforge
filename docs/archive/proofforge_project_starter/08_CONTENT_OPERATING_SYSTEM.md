# Content Operating System

## Publishing Unit
One portfolio item is the atomic unit.

## Definition of Done for One Item
An item is publishable only when it has:
- canonical metadata
- thumbnail or capture
- detail page
- clear promise
- clear business value
- explicit status
- explicit protection level

## Acceptable Item Types
- mini-app
- prompt lesson
- prompt system
- code recipe
- best-practice note
- UI pattern
- data workflow
- product teardown
- micro utility
- repeatable template

## Publishing Quality Filter
Reject any item that is:
- vague
- redundant
- hard to explain in one sentence
- visually unproven
- commercially irrelevant
