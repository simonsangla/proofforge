# ProofForge

## One-line Description
ProofForge is a portfolio PWA built as a continuous proof-of-work system for publishing 5 to 7 high-signal portfolio items each week.

## Strategic Purpose
ProofForge is not a cosmetic portfolio. It is a structured publishing system designed to:
- turn regular creation into durable proof of capability
- showcase public demos without exposing core proprietary logic
- organize replicable assets and premium/private engines
- compound trust, signal, and commercial leverage over time

## Core Rules
- Portfolio shell first, but never as an empty shell
- Every item must follow one canonical artifact contract
- Public demo is allowed; core value logic stays private
- The system must stay simple enough for weekly shipping
- Design must remain elegant, minimal, and non-blue

## Weekly Publishing Target
- 5 to 7 new portfolio items per week
- 1 to 2 can be mini-apps
- the rest can be prompt systems, code patterns, recipes, UI patterns, micro-tools, analyses, or reusable assets
